Official website for the XJTLU Microsystems Research Group, led by Prof. Pengfei Song, focusing on micro/nano technology, biosensors, microfluidic platforms, and automation control.

## Quick Start

### Prerequisites

- Node.js >= 18.17
- npm or yarn

### Installation

```bash
git clone <repository-url>
cd research_group_website
npm install
```

### Development

```bash
npm run dev
# Visit http://localhost:3000
```

### Build

```bash
npm run build
npm start
```

## Tech Stack

- **Framework**: Next.js 15.5.6 (App Router)
- **UI**: React 19
- **Styling**: Tailwind CSS 3.4
- **Language**: TypeScript 5
- **Deployment**: Vercel

## Deployment

Deploy to Vercel:

```bash
npm i -g vercel
vercel
```

Or connect your GitHub repository to [Vercel](https://vercel.com) for automatic deployments.
